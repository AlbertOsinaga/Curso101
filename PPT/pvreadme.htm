Acerca de Visor de PowerPoint

Visor de Microsoft Office PowerPoint permite ver presentaciones creadas en Microsoft Office PowerPoint 2003 con toda su funcionalidad y características; además, es compatible con las presentaciones creadas con PowerPoint 97, PowerPoint 2000 y PowerPoint 2002. Asimismo, Visor de PowerPoint admite la apertura de presentaciones de PowerPoint protegidas con contraseña.

Requisitos del sistema

  • PC con un procesador a 200 MHz o superior
  • Microsoft Internet Explorer 5 o versión posterior
  • Sistemas operativos compatibles: Microsoft Windows 98 Segunda edición, Windows Millennium, Windows 2000 con Service Pack 3, Windows XP o Windows Server 2003.
  • RAM (Random access memory):
    • Para Windows 98 SE, Windows ME: 32 MB de RAM
    • Para Windows 2000 con Service Pack 3: 64 MB de RAM
    • Para Windows XP, Windows Server 2003: 128 MB de RAM
  • 5 MB de espacio de disco duro disponible
  • Pantalla Super VGA de 800X600 o con resolución superior

Problemas relacionados con los idiomas

Visor de PowerPoint no es compatible con las Herramientas de Corrección de Microsoft Office.

Visor de PowerPoint no es compatible con las presentaciones en idiomas bidireccionales creadas con PowerPoint 97.

Trabajar con Visor de PowerPoint

Visor de PowerPoint no muestra correctamente las animaciones de diagramas. En lugar de mostrar las animaciones, el diagrama aparecerá completo.

Visor de PowerPoint no permite abrir presentaciones de Administración de permisos sobre la información (IRM).

Visor de PowerPoint no admite la ejecución de macros o programas ni la apertura de objetos vinculados o incrustados.

Visor de PowerPoint no es capaz de mostrar los siguientes formatos de gráficos cuando están vinculados a una presentación: .eps, .pct, .emz, .cdr, .cgm, .wpg y .wmz.
